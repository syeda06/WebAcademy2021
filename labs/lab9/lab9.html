Inline CSS-

Inline CSS allows us to apply style rules to our HTML documents. Inline CSS can be best used when you want to preview a change or style only a few elements on a web page. Inline CSS is helpful for when you want to develop and style your HTML documment by adding fonts, color, e.t.c.

Internal CSS-

Internal CSS is writeen in the "head" section of the HTML documment. With the internal CSS, we can add styles to our documment with the "style" command. the Internal CSS is used to define a style for a single HTML page.

External CSS-

External CSS is a separate CSS file that can be accessed by creating a link within the head section of the webpage. We can connect multipule webpages with the link to the stylesheet to apply the same styles to different webpages. the external stylesheet is places in the head section of the HTML documment or webpage.
